`ssh random@pwnable.kr -p2222 (pw:guest)`

获得源码如下：

```c
#include <stdio.h>

int main(){
	unsigned int random;
	random = rand();	// random value!

	unsigned int key=0;
	scanf("%d", &key);

	if( (key ^ random) == 0xdeadbeef ){
		printf("Good!\n");
		system("/bin/cat flag");
		return 0;
	}

	printf("Wrong, maybe you should try 2^32 cases.\n");
	return 0;
}
```

发现rand没设种子，也就说每次生产的随机数都是一样的，所以输入一个与之异或值为 0xdeadbeef 的数就行了。
